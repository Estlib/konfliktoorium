Visual Studio Code

Mis see on?

Visual Studio Code on kerge, kuid võimas lähtekoodiredaktor, mis töötab teie töölaual ja on saadaval Windowsi, macOS-i ja Linuxi jaoks. Sellel on sisseehitatud JavaScripti, TypeScripti ja Node.js-i tugi ning sellel on rikkalik laienduste ökosüsteem muude keelte ja käitustingimuste jaoks (nt C++, C#, Java, Python, PHP, Go, .NET).

Visual Studio

  • Vahendi nimi: Visual Studio
  • Vahendi tüüp: Integreeritud arenduskeskkond (IDE)
  • Platvorm: Windows. Väljalaskeaeg: 1997
  • Eesmärk: Visual Studio on terviklik tarkvaraarenduse tööriistakomplekt, mis võimaldab arendajatel luua erinevaid rakendusi, sealhulgas veebi-, mobiili- ja pilvetarkvara.
  • Tutvustus: Visual Studio pakub laia valikut funktsioone, sealhulgas terviklikku integreeritud arenduskeskkonda, koodiredaktorit, silumis- ja testimistööriistu ning meeskonnatöö funktsioone.
  • Kasuks ja vastu:
    Kasuks Vastu
    Terviklik tugi mitme keele ja platvormi jaoks Aeglane ja ressursinõudlik
    Integreeritud arendus-, silumis- ja testimistööriistad Mõned funktsioonid võivad olla keerulised algajatele
    Meeskonnatöö funktsioonid (näiteks Git integreerimine) Litsentsitasu võib olla kõrge
  • Salespitch: Visual Studio pakub kõikehõlmavat lahendust tarkvaraarenduseks, mis võimaldab arendajatel tõhusalt ja produktiivselt luua mitmesuguseid rakendusi igas suuruses ja keerukusastmes.
  • Näide: Näiteks võib õpilane kasutada Visual Studio't C# programmeerimiskeele ja ASP.NET raamistiku abil veebirakenduse loomiseks ning seda integreerida Azure'iga pilvepõhise majutusteenuse kasutamiseks.
