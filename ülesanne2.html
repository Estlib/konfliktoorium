Visual Studio Code

Mis see on?

Visual Studio Code on kerge, kuid võimas lähtekoodiredaktor, mis töötab teie töölaual ja on saadaval Windowsi, macOS-i ja Linuxi jaoks. Sellel on sisseehitatud JavaScripti, TypeScripti ja Node.js-i tugi ning sellel on rikkalik laienduste ökosüsteem muude keelte ja käitustingimuste jaoks (nt C++, C#, Java, Python, PHP, Go, .NET).
